# Exploratory Data Analysis on Pokémon Stats Dataset

## Overview
This repository contains an Exploratory Data Analysis (EDA) project
performed on a Pokémon statistics dataset.

## Dataset
- Source: Kaggle
- File: pokemon_stats_2025.csv
- Description: Contains physical attributes, battle stats, and types of Pokémon.

## Contents
- data/: Dataset used for analysis
- notebooks/: Jupyter Notebook with EDA and visualizations

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Strong correlation between attack and special attack
- Defense and special defense are closely related
- Missing values in type_2 indicate single-type Pokémon

## How to Run
1. Clone the repository
2. Install dependencies
3. Open the notebook and run all cells
